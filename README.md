# go-fiber

## installation

- install go
- install glide
- install gomon or gin or go reload ....
- export path

  ```
  export GOPATH=$(go env GOPATH)
  export GOBIN=$GOPATH/bin
  export PATH=$PATH:$GOBIN
  ```

- use glide instead go Ex. go get ... to glide get ...

## run

- go run server.go or gomon
